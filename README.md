# MyCadStarter

Небольшое учебное CAD-приложение на **Qt Widgets** и **OpenCascade**.

## Что добавлено

- более аккуратный интерфейс с тёмной темой, иконками и статусной строкой;
- новые операции: создание **конуса** и **тора**;
- отдельные команды вида и камеры: **изометрия**, **Fit All**, **показать/скрыть лог**;
- подключение **GoogleTest** через `FetchContent`;
- базовые unit-тесты для вспомогательной логики;
- конфигурация **Doxygen** и цель `docs` для генерации документации.

## Сборка приложения

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск тестов

```bash
ctest --output-on-failure
```

## Генерация документации

Требуется установленный `doxygen`.

```bash
cmake --build . --target docs
```

После генерации HTML-страницы будут находиться в `docs/build/html`.
